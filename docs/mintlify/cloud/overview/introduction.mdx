Chroma Cloud is a fast, scalable, serverless database-as-a-service built on top of open-source Chroma. It gives you everything you need for retrieval — dense, sparse, and hybrid vector search, metadata filtering, and multi-modal retrieval — with zero operations. [Sign up for free](https://trychroma.com/signup?utm_source=docs-cloud-introduction) and get $5 in credits.
